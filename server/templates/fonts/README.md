Template fonts used by the Fairyteller PDF renderer.

- Rubik Mono One: Google Fonts / SIL Open Font License 1.1
- Amatic SC: Google Fonts / SIL Open Font License 1.1
- Inter static TTFs: rsms/inter release v4.1 / SIL Open Font License 1.1

The renderer embeds these files so production PDF output is repeatable and does not depend on VPS system fonts.
